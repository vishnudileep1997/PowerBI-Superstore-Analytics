# Superstore Sales Dashboard – Power BI Project

## Project Overview
This project is an interactive Power BI dashboard developed to analyze Superstore sales performance, profit trends, customer segments, and regional business insights.

The dashboard helps identify:
- Sales trends
- Profitability analysis
- Regional performance
- Customer behavior
- Product category performance

---

## Tools & Technologies
- Power BI
- Excel / CSV Dataset
- DAX
- Data Visualization

---

## Key KPIs
- Total Sales
- Total Profit
- Quantity Sold
- Profit Margin
- Regional Performance
- Category-wise Analysis

---

## Dashboard Features
- Interactive Filters & Slicers
- Sales Trend Analysis
- Region-wise Performance
- Product Category Insights
- Profitability Dashboard
- Customer Segment Analysis

---

## Business Insights
- Identified top-performing regions
- Analyzed high-profit product categories
- Tracked monthly sales growth
- Compared sales vs profit trends
- Evaluated customer segment contribution
